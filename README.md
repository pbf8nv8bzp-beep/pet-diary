# 🐾 萌宠健康日记 (Pet Health Diary)

一个可爱、轻量级的宠物健康记录 Web App。基于纯 HTML/CSS/JS 开发，无后台数据库，数据存储在本地浏览器中。支持 PWA，可直接添加到手机桌面作为 App 使用。

---

## 🚀 如何使用

### 在线访问
访问 GitHub Pages 地址： `https://pbf8nv8bzp-beep.github.io/pet-diary/`

### 安装到手机 (PWA)
1. 在手机浏览器 (Safari/Chrome) 中打开网页。
2. 点击浏览器的“分享”或“菜单”按钮。
3. 选择 **“添加到主屏幕”**。
4. 即可像原生 App 一样离线使用。

## 🛠️ 技术栈
- HTML5
- CSS3 (Flexbox, CSS Variables, Dark Mode)
- JavaScript (ES6+, LocalStorage, Chart.js)

## ⚠️ 注意事项
本项目默认使用浏览器的 `localStorage` 存储数据。**在清理浏览器缓存前，请务必使用页面底部的“导出备份”功能保存数据！**
